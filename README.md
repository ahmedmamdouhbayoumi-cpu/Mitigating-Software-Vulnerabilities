# Mitigating Software Vulnerabilities

This project demonstrates and provides mitigation strategies for common web application vulnerabilities, specifically Cross-Site Scripting (XSS) and SQL Injection (SQLi).

---

## Project Goals

* **Reflected XSS Attack:** Understanding and executing a Cross-Site Scripting attack.
* **SQL Injection Attack:** Demonstrating an SQL Injection attack on a login page.
* **Mitigation Strategies:** Proposing effective steps to prevent XSS and SQLi vulnerabilities.
